Configuracion inicial del Software Pydroponia¶

Una vez que el sistema se ha montado, segun las instrucciones en Montaje del sistema, la tarjeta micro SD esta en la posicion correcta y la fuente de poder conectada a la Raspberry Pi, procedemos a energizar el sistema.

** IMPORTANTE ** asegurarse que la fuente de poder (Item #3 en Tabla 1) tiene el adaptador correcto para el tipo de enchufe.

** Esta seccion es relevante solo si se necesita re-instalar o re-configurar el software. El sistema se entrega con el software ya configurado y listo para utilizar en la tarjeta micro SD (Item #4 en Tabla 1).**
